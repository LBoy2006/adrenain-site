Политика конфиденциальности

Настоящая Политика определяет порядок обработки и защиты персональных данных пользователей сервиса «Адреналин Фэмили» (далее — Сервис), доступного в Telegram-боте t.me/adrenalinfamilybot и на сайте adrenalinfamily.ru.

1. Общие положения

1.1. Настоящая Политика определяет порядок обработки и защиты персональных данных пользователей Сервиса.

1.2. Используя Сервис, пользователь подтверждает согласие с условиями настоящей Политики.

2. Какие данные мы собираем

2.1. Данные, предоставленные самим пользователем:

  • Имя, никнейм, аватар, адрес доставки;
  • Контактные данные (email, телефон);
  • Платёжная информация (обрабатывается платёжным провайдером).

2.2. Автоматически:

  • Telegram ID;
  • Информация об использовании Сервиса (действия, внутренняя статистика).

3. Для чего мы используем данные

  • Для регистрации и авторизации в Сервисе;
  • Для начисления виртуальных ресурсов и доставки призов;
  • Для проведения акций и розыгрышей;
  • Для обратной связи и технической поддержки;
  • Для исполнения требований закона РФ.

4. Передача данных третьим лицам

4.1. Мы можем передавать данные только:

  • Платёжным провайдерам — для обработки оплаты;
  • Логистическим компаниям — для доставки призов;
  • По запросу госорганов, если это предусмотрено законом.

5. Хранение и защита данных

5.1. Данные хранятся на защищённых серверах.

5.2. Мы принимаем все необходимые организационные и технические меры для защиты данных.

6. Срок хранения

6.1. Данные хранятся до достижения целей обработки или по запросу пользователя на удаление.

7. Контакты

По вопросам обработки данных: info@adrenalinfamily.ru
